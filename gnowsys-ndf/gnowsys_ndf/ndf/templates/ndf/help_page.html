This is the help section detailing features and frequently asked questions (FAQs) of CLIx Platform.

https://clixplatform.tiss.edu

Release: April 2017

 

Student FAQs

 

What is My Desk?

My Desk is the place where students will land after logging in to CLIx platform. In My Desk, My Courses and My Performance will be displayed. For more, read My Courses and My Performance.

 

What is My Courses?

My Courses displays all the In-Progress and Completed CLIx units of a student.

 

What is My Performance?

It shows the following details about the student's work on the CLIx platform: progress of the enrolled units, lessons, assessments, file uploads, interactions, note making and total points earned. That is, My Performance is an aggregated dashboard of how an individual student is progressing in the CLIx learning journey. For more, read My Analytics.

 

 

What is Progress Report?

Progress Report refers to the CLIx unit level performance of  a student that include: lesson progress, assessments, file uploads, interactions, note making and total points earned.

 

 

What is Explore?

Explore is the section that contains all the CLIx Modules and Units in which students can enrol. For more, read CLIx Module and CLIx Unit.

Explore is the button which students can click on to find out what all is available in the CLIx platform. By clicking on Explore, the student will be able to see all CLIx modules and CLIx units into which they can enrol. For more, read CLIx Module and CLIx Unit

 

 

What is a CLIx Module?

A CLIx Module is a set of related CLIx Units. For more please read CLIx Units

 

What is a CLIx Unit?

CLIx Unit  is a course where students can enroll or register. A CLIx Unit has many Lessons that in turn contain Activities. For more, read CLIx Lesson and CLIx Activity.

 

 

What is a CLIx Lesson?

Many CLIx Lessons together make up a CLIx Unit. A Lesson is made up of Activities. Each lesson is designed to be completed in one session. For more, read CLIx Activity.

 

 

What is a CLIx Activity?

A CLIx Activity is what student can "do" on the CLIx platform. A student will be able to do different types of activities like reading a text, watching a video, listening to a story, working on a quiz, writing a note etc., on the CLIx platform. An activity can be in the form of text, video, image, audio, assessment, quiz or a combination of these.

 

 

Who is a Buddy?

Buddy is a peer student who shares the computer with one or more other buddies during CLIx learning session. Buddies help each other and work with each other during a CLIx period.. Buddies learn together.

Buddies can be added or removed during a CLIx session as per the teacher’s guidance.

 

What are Resources?

Resources are files such as images, audios and videos that will be readily available for students. Students can download these resources and make use of them to  complete activities during the CLIx sessions.

 

 

What is Notebook?

Notebook allows teachers and students to post their notes, see others’ notes  and discuss with each other by reply option.

 

What is Gallery?

Gallery allows students to upload files such as images, audios, videos and documents. For more, read How to upload on gallery.

 

What are Interactions?

Interactions are the replies you get on your notebook posts and the replies you post on others notebook.

 

How is CLIx Curriculum is structured?

CLIx curriculum is structured as: Modules -> Units -> Lessons -> Activities.

Modules contain units; Units contain lessons; Lessons contain activities, and; Activities contain text, images, audios, videos, tools, games and assessments.

 

How to enrol in a Unit?

1.      Login to the CLIx platform.

2.      Go to Explore where all the modules and units are displayed.

3.      Click on a module. All the units within that module will be displayed.

4.      Click on a unit. The unit overview will be displayed.

5.      On the right hand side of the image click on the Enrol button. The button will turn green and status will be changed to Enrolled.

6.      Go to My Desk, in the My Projects the enrolled unit will be displayed.

7.      Click on the unit and start enjoying the CLIx learning journey!

 

Why should a student enrol in a Unit?

Enrollment in a unit is required to get the points when you complete lessons, upload files or make notes. If you do not enroll for a unit, then you will not be able to get any points and know the progress you make. Hence, never forget to enroll in the units :)

 

How to upload files on the gallery?

Gallery allows file upload in an organized way. In order to upload files, first a folder needs to be created. Within the folder files can added as below.

1.      Go to gallery.

2.      Click on the Add button.

3.      Enter the name of folder and briefly describe what kind of files this folder will contain.

4.      Add tags if any (for more details on tags see what is a tag?)

5.      Save the folder.

6.      Once the page is refreshed and folder is generated, click on the folder.

7.      Click on the Add File button.

8.      Enter file name.

9.      Enter description (if any).

10.  Choose a file by pressing on ‘Select File’

11.  Click on ‘Upload’ button, the file will be saved.

12.  You can add alternate files using the Link Related File.

 

How to gain points?

Points are given for your learning progress. Below are some of the ways in which you can gain more and more points:

1.      Enrol into a unit

2.      Complete all the lessons and activities

3.      Take assessments & quiz

4.      Make notes

5.      Discuss on the notes created by other students

6.      Upload files in the gallery

7.      Make use of Resources

8.      If others give you ratings (stars)

Teacher FAQs

What is My Desk?

My Desk is the section that is unique to every login user. My Desk contains two more sub-sections: My Projects and My Performance. Check My Projects and My Performance to know more.

 

What is My Courses?

My Courses displays all the In-Progress and Completed CLIx units of a login user.

 

What is My Performance?

My Performance displays unit level performance for all the students enrolled across the CLIx units. It shows the analytics pertaining to each student. That is, My Performance is an aggregated dashboard of how an individual student is progressing in the CLIx learning journey. For more, read Progress Report.

 

 

What is Progress Report?

Progress Report for teacher will display the unit level performance of  all the students enrolled in that unit. The details include unit and lesson completion status, total points gained, gallery upload, note making and interactions.

If you click on a student’s name, the detailed view of that student will be displayed. The detailed view will be same as what the students will themselves see with their login.

 

What additional privileges/accesses a teacher will have on the CLIx platform?

 

A teacher will have some additional features than the students:

1.      View all students’ progress

2.      View teacher help page in an activity

3.      Edit and delete students’ notes

4.      Edit and delete comments made by students

5.      Upload files in Resources

6.      Delete file uploads by students in the gallery

7.      Reset student's login password

 

What is Teacher Page?

Teacher Page is the activity level help page available for the teachers. Any specific instructions related to that particular activity will be provided for teachers in teacher page.
